# StreamDrop

The final, polished software version of my YouTube Downloader — now installs seamlessly on Windows with zero setup required.

No Python. No FFmpeg. No command line. Just download, install, and use.

---

## How to Install

1. Download `StreamDropSetup.exe` from this repository
2. Double-click it to run the installer
3. Follow the setup wizard (Next → Next → Install)
4. Choose your install location when prompted
5. Done! StreamDrop is now installed with a Desktop icon and Start Menu entry

## Usage

Just double-click the StreamDrop icon on your Desktop anytime to open the app — paste a YouTube link, choose your format and quality, and download.

## Features

- Download YouTube videos up to 4K
- Extract MP3 audio (128–320kbps)
- Download full playlists automatically
- Live progress bar with speed & ETA
- Cancel any download mid-way
- Opens as a native-feeling desktop app — no browser tabs, no address bar
